Katydid Sample Applications

  1. Hello World The simplest possible Katydid-VDOM usage; builds the virtual DOM only once.
  2. Greet Me A bit more complicated, incorporating user events to change the greeting.
  3. Sudoku Solver A Sudoko solver illustrating an application lifecycle and Katydid usage with more complicated views and updates plus use of Katydid-CSS for its stylesheet.
  4. Digital Clock A simple digital clock illustrating the addition of commands to a Katydid lifecycle.
  5. WIP Cards TO DO: A very simple kanban board.
